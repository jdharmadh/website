data structures

I like testing about theoretical algorithms on big datasets. Particularly, I'm interested in sublinear sketches, streaming algorithms, and hashing.
Probabilistic Sketches Bloom Filter, HyperLogLog, CountMin
Solving problems like keeping a counter or marking distinct elements is easy —unless you're Google or Meta and the number of data points you have is in the billions. Additionally, how do you run algorithms designed for arrays when all of your input doesn't even fit in memory? I provide implementations of common data structures that solve these problems by sketching, i.e. sacrificing a small amount of accuracy for tons of efficiency. Take a look here.
Augmented Trees BinHeap, vEB Tree, Interval Tree
Binary search trees are a great general-purpose data structure that every freshman learns about. However, if we make some small modifications to how they store data, we can solve a much wider range of problems efficiently. Take a look here.
